# AI Study Assistant

A full-stack AI-powered study assistant using React, Node.js, MongoDB, and OpenAI API with Retrieval-Augmented Generation (RAG).

## Quick Start

### Backend
```bash
cd server
npm install
cp .env.example .env
# Edit .env with your credentials
npm run dev
```

### Frontend
```bash
cd client
npm install
cp .env.example .env
npm start
```

## Documentation

See [README.md](./README.md) for complete documentation, setup instructions, and API reference.

## Features

- 📤 Upload PDF and DOCX files
- 🤖 Chat with AI based on your documents
- 📝 Auto-generate summaries
- 📋 Create quizzes and flashcards
- 🔐 Secure authentication with JWT
- 🎨 Modern, responsive UI

## Tech Stack

- Frontend: React, Tailwind CSS
- Backend: Node.js, Express, MongoDB
- AI: OpenAI API, RAG implementation
